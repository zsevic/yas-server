# yas-server

> Back-end for [Yet Another Schedule](https://github.com/zsevic/yas)

### :wrench: Setup

```bash
git clone https://github.com/zsevic/yas-server.git
cd yas-server
npm i
npm run dev
```

### :construction_worker: Build

```bash
npm run build
npm start
```

### :rotating_light: Testing

```bash
npm test
```

### :package: Technologies used

* Node.js, Express
